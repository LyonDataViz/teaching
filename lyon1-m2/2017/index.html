Cours visualisation de données

Université Lyon 1, Master 2, 2017-2018

Objectifs du cours

Ce cours est une introduction au domaine de la Data Visualization

  • Principes de base en perception, congition et visualisation
  • Connaitre la typologie des graphiques pour les principaux types de données
  • Réaliser une visualisation Web interactive en utilisant d3.js en utilisant un jeu de données réel
